Shooter
=======

YAV of the canonduin/intervaluino camera intervalometer. Code and PCB.
Mini Arduino shield format.

Connect into pins GND..9.


Pinout

1 -> GND
2 -> LED 1
3 -> Focus
4 -> Shoot
5 -> Button Input
6 -> LED 2



WiShield
--------

Connect into pins 7..3.

Check the 'widow' folder for the wifi enabled version.
